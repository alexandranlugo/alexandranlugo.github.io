Altristotle: AI Learning Assistant for Underfunded Schools

LLM Development Product Design Social Impact Backend Development

The Problem: Students in underfunded schools lack access to personalized tutoring, but giving AI-generated answers doesn't build critical thinking skills.

The Solution: I developed Altristotle, a customized Large Language Model that acts like a Socratic tutor—asking guiding questions instead of providing direct answers. This helps students develop problem-solving skills independently.

Technical Implementation: Built during a 6-week MIT RAISE x SureStart program, using Ollama for backend LLM customization and a WebUI interface. Deployed locally on GPU-supported environments for efficient real-time interactions. Competed in the Create-a-Thon competition.

Impact: Designed specifically for students who need it most—those without access to expensive tutoring services. Demonstrates how AI can be a force for educational equity when built thoughtfully.

Key Skills: LLM customization, prompt engineering, backend development (Ollama), user-centered design, GPU optimization
